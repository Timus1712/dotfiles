# My dotfiles

Very boring configs for:
* zsh (oh-my-zsh)
* vim
* awesome
* git
* st (simple terminal)

# How it looks
![vim](example_images/vim.png?raw=true)
![zsh](example_images/zsh.png?raw=true)

You can install all configs:
```
git clone --recursive https://github.com/Timus1712/dotfiles.git
make
```
Or specified config:
```
git clone --recursive https://github.com/Timus1712/dotfiles.git
make vim
```
